# Open Source Legal

**Building open source tools for understanding how law, contracts, and legal systems shape our world.**

---

Contracts govern nearly every meaningful transaction in modern life — employment terms, rental agreements, insurance policies, software licenses. Yet the tools for understanding, comparing, and analyzing them have remained locked behind commercial paywalls.

We believe the quiet, compounding power of open source can change that. One tool at a time. One knowledge base that anyone can fork, annotate, and extend.

### The Knowledge Distillation Funnel

Raw legal material passes through progressive layers of refinement — from unstructured natural language to version-controlled collections, structured data, AI-assisted curation, and finally into searchable, visual, actionable presentations.

```
NATURAL LANGUAGE                 Raw contracts, regulations, statutes
    ↓
VERSION CONTROLLED COLLECTIONS   Versioned corpora, forks, document stores
    ↓
STRUCTURED DATA                  Annotations, schemas, extracted fields
    ↓
AUTOMATION & CURATION            AI agents + human-in-the-loop review
    ↓
PRESENTATION                     Viewers, redlines, dashboards
    ↓
DISTILLED INSIGHTS & ANSWERS
```

### Projects

| Project | Description |
|---------|-------------|
| **[OpenContracts](https://github.com/Open-Source-Legal/OpenContracts)** | Self-hosted document annotation, version control, semantic search, and MCP integration. The platform at the center of the stack. |
| **[Docxodus](https://github.com/JSv4/Docxodus)** | Office XML redline engine — compares Word documents, detects moved sections, converts DOCX to HTML. Runs in the browser via WebAssembly. |
| **[react-docxodus-viewer](https://github.com/JSv4/react-docxodus-viewer)** | Drop-in React component for viewing DOCX documents entirely in the browser. Powered by Docxodus WASM. |
| **[Python-Redlines](https://github.com/JSv4/Python-Redlines)** | DOCX tracked-change redlines for the Python ecosystem. Cross-platform binaries for Linux, macOS, and Windows. |
| **[CAML](https://github.com/Open-Source-Legal/caml)** | Corpus Article Markup Language — a human-readable markdown superset for authoring interactive legal articles and knowledge bases. |

### The Conviction

Software is a necessary but not sufficient condition for innovation in the governance layer of our societies. The real value lives in human expertise, judgment, and preference — the people who understand what a clause means, why a regulation matters, and how a policy should evolve. No amount of tooling replaces that.

The open source legal stack exists to free you from the software problem so you can focus on what actually matters. Stop rebuilding document parsers. Stop licensing comparison engines. Stop fighting your infrastructure. The plumbing should be solved, shared, and invisible — so that every hour of human attention goes toward the work that only humans can do.
